# Smart Temperature Monitoring System

## Overview
This project is a C++ embedded-style application designed to simulate a smart temperature monitoring system.
The architecture follows STM32-style modular design principles.

## Features
- Modular C++ design
- Simulated temperature sensor
- Alarm mechanism for over-temperature conditions
- Embedded-friendly structure

- C++
- Embedded Systems Concepts
